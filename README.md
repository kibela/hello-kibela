# Hello, Kibela Web API!

This is a sample project to call [Kibela](https://kibe.la/) Web API.

## Prerequisites

NodeJS v10 or greater.

## Setup

```shell-session
# Install dependencies
npm install

# Configure KIBELA_TEAM and KIBELA_TOKEN
code .env

# Run a simple script
npx ts-node hello-kibela.ts
```
